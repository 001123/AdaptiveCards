# Getting started - UWP

This is a renderer which targets UWP native controls.

## Install NuGet package

```console
Install-Package AdaptiveCards.Rendering.Uwp -IncludePrerelease
```

## Namespace

Declare the renderer namespace.

```csharp
using AdaptiveCards.Rendering.Uwp;
```
